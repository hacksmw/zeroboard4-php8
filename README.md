# zeroboard4-php8
zeroboard4-php8를 기반으로 리팩터링 한 것 입니다.

## 변경내역
* password, old_password, sha1 등의 함수가 없는 MySQL 8 이상 버전에서 작동.
* HTML 태그를 문법에 맞게 수정. 레거시 테이블 레이아웃은 아직 바꾸지 않았습니다.
* extract를 사용하지 않고 명시적으로 파라미터 값을 읽어오도록 수정하였습니다. 스킨 등의 호환성을 위해 extract는 아직 남겨두었습니다.
* 추후에 사용되지 않은 변수를 읽을 때 오류가 발생하지 않도록 php9 호환성 보장 예정.
* 파라미터에 htmlspecialchars, urlencode 같은 것을 적용하여 XSS나 버그 방지.
* 관리자 페이지는 referer 체크로 admin.php를 경유해야 접근할 수 있도록 수정하였고 추후 admin.php에서 관리자 암호를 요구하게 하여 CSRF에 대한 1차적인 방어를 할 예정.

##
버그와 취약점을 없애려고 노력했지만 결함이 있을 수 있습니다. 이에 대한 손해는 책임지지 않습니다.
스킨을 포함한 기존 버전과의 호환성은 보장하지 않습니다.
